# 📌 Project Overview

This project presents an **Exploratory Data Analysis (EDA)** of the Student Performance dataset using Python. The objective is to identify the key factors influencing students' academic performance through statistical analysis and data visualization.

The analysis examines how study habits, previous academic failures, parental education, internet access and other academic and demographic factors relate to students' final grades. Various visualization techniques were used to uncover meaningful insights and support data-driven educational decision-making.

# 🔑 Key Insights

- Most students achieved moderate academic performance, with an average final grade of **10.42**.
- Students studying **5–10 hours per week** obtained the highest average final grades.
- Previous academic failures showed a strong negative relationship with academic performance.
- Students whose parents had higher education generally performed better academically.
- Internet access showed a slight positive association with students' final grades.
- First and second period grades (**G1** and **G2**) were the strongest predictors of the final grade (**G3**).

---

# 💡 Recommendations

- Encourage students to maintain consistent study habits.
- Provide early academic support for students with previous failures.
- Improve access to digital learning resources.
- Promote greater parental involvement in students' education.
- Use early academic performance indicators to identify students who may benefit from additional support.

# ✅ Conclusion

This project demonstrates how Exploratory Data Analysis can uncover meaningful factors influencing student performance. The findings highlight that study habits, previous academic failures, parental education, internet access and earlier academic performance all contribute to students' final grades. These insights can support educators in making informed, data-driven decisions to improve academic outcomes.
